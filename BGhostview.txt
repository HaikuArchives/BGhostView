Compatibility: R4
     Location: Graphics
  Description: Postscript viewer
        Notes: BGhostview is a viewer for Postscript and PDF-Files. It provides
               a GUI which allows to access random pages, and save them
               to a new postscript-file-
               Jake Hamby's BeOS-Port of Aladdin Ghostscript is required. Make
               sure the gs-libraries are in the library-path (see readme for details).
               libprefs and XPMTranslator are also required.
               Full source is included. Thanks to Jake Hamby for helpful advice.
